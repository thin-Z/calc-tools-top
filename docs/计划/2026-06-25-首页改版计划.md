### 任务 5：`js/i18n.js` — 更新语言切换跳转目标

**文件：**
- 修改：`calculator-site/js/i18n.js`

- [ ] **步骤 1：修改 `switchLang` 函数**

```javascript
function switchLang(lang) {
    localStorage.setItem("lang", lang);
    if (lang === "zh") {
        window.location.href = "/";  // 原为 "/zh/"
    } else {
        window.location.href = "/en/";
    }
}
```
