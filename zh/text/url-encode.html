什么是 URL 编码？

                
URL 编码（百分号编码）是在特定情况下对 URI 中的信息进行编码的机制。它将特殊字符和非 ASCII 字符转换为百分号（%）后跟两个十六进制数字，代表该字符的字节值。这确保数据能通过 HTTP 安全传输而不破坏 URL 结构。

                
本工具使用浏览器内置的 encodeURIComponent 和 decodeURIComponent API 进行 UTF-8 安全的编码与解码。所有处理均在浏览器本地完成，数据不会离开你的设备。


                
常见使用场景

                
    
                    
  • GET 请求参数：编码查询参数以安全传输特殊字符和非 ASCII 文本
    • 
                    
  • POST 表单数据：application/x-www-form-urlencoded 格式需要 URL 编码
    • 
                    
  • AJAX / Fetch 请求：构建带编码参数的安全 API 请求 URL
    • 
                    
  • URL 重定向：对重定向 URL 中的目标地址进行编码
    • 
                    
  • Cookie 值编码：对特殊字符编码以安全存储 Cookie
    • 
                


                
重要说明

                
与 encodeURI 不同，encodeURIComponent 会编码所有非字母数字字符，包括 URL 结构中合法存在的字符（如 : // ? # &）。本工具使用 encodeURIComponent / decodeURIComponent，这是编码查询参数值时的正确选择。仅当你需要编码完整 URL 且不破坏其结构时，才使用 encodeURI。
