Last updated: June 28, 2026

Our Promise: All tools process your data locally in the browser. Files never leave your device. We do not collect any personal data when you use our tools.

1. Information We Collect

ToolBox Inside (calc-tools.top) is a fully static tool website. All tools run entirely in your browser:

  • Usage Data: None collected. All calculations, image processing, and text processing happen locally.
  • Personal Information: None collected. No registration or login required.
  • File Uploads: Files never leave your device. Image tools use Canvas API for local processing.

  • Cloudflare (CDN & security): May set security cookies for CDN and DDoS protection.

4. Data Security

Since all tools process data locally in your browser, there is no data transmission risk. However, the security of any data you store locally depends on your browser and device protection measures.
