## 一、项目概述

构建一个**中英双语的在线计算器聚合站**，首批上线 5 个工具，为后续扩展为站群旗舰站奠定架构基础。

### 核心目标
- MVP 在 4 天内上线
- SEO 友好，每个工具独立 URL
- 中英双语独立路径（/zh/、/en/）
- 纯前端实现，零后端依赖
- 预留广告位，为 AdSense 申请做准备

---

## 二、技术架构

### 2.1 架构方案（选定：多页静态站）

采用"多页静态 HTML + 独立 JS 文件"架构，每个工具一个独立页面。

### 2.2 项目结构

### 2.3 技术选型

| 项 | 方案 | 理由 |
|---|------|------|
| 语言 | 纯 HTML + CSS + JavaScript | 零依赖，可部署在任何静态托管 |
| 部署 | Vercel / Cloudflare Pages | 免费，GitHub 一键部署 |
| 版本管理 | GitHub | 免费，CI/CD |
| 域名 | DNS → Cloudflare | 免费 CDN + SSL |

## 三、工具功能规格

### 3.1 房贷计算器（Mortgage Calculator）

**中文搜索热度**: 🔥🔥🔥🔥🔥
**中/英**: 🇨🇳

**输入参数**:
- 贷款总额（元）
- 年利率（%）
- 贷款年限（年）
- 还款方式：等额本息 / 等额本金

**输出结果**:
- 月供金额
- 还款总额
- 利息总额
- 利率表（逐年）

**可视化**: 还款比例饼图、逐年还款曲线图

**广告位**: 结果上方 + 侧边栏

---

### 3.2 个税计算器（2026 Tax Calculator）

**中文搜索热度**: 🔥🔥🔥🔥
**中/英**: 🇨🇳

**输入参数**:
- 税前月薪（元）
- 社保公积金扣除（元）
- 专项附加扣除（元）
- 年终奖（可选）

**输出结果**:
- 应纳税所得额
- 适用税率

- 税后月薪

**可视化**: 税率分段展示、税前税后对比

**广告位**: 结果上方 + 侧边栏

## 四、SEO 配置

### 4.1 每页必须包含
- 唯一 title + meta description
- Open Graph 标签
- hreflang 标签（中英文对应）
- JSON-LD 结构化数据（SoftwareApplication）
- Canonical URL

### 4.2 URL 策略
- 中文: /zh/mortgage/ → 对应页面
- 英文: /en/mortgage/ → 对应页面
- 首页: /zh/、/en/

### 4.3 广告位预留
- 首屏上方（广告位 A）
- 结果区域下方（广告位 B）
- 侧边栏（广告位 C）
- 移动端插屏位置（广告位 D）
- 先用 <div class="ad-slot" data-slot="A"></div> 占位

## 六、与后续站群架构的兼容性

| 后续扩展 | 本架构的准备 |
|---------|------------|
| 城市长尾词页（/bj/mortgage/） | 新增 HTML 页，复用计算 JS |
| 单工具独立站 | 每个工具 JS 独立文件，直接搬走 |
| SEO 内容文章 | 直接在 zh/en 下新增内容页 |
| 新语言（日语/韩语） | 新增 /ja/ /ko/ 目录 |
| Freemium PDF 导出 | 在工具页新增功能，不影响其他页 |
| 通用模板脚本 | scripts/generate.js 从模板批量生成 |
