About ToolBox Inside

ToolBox Inside is a free collection of online tools including calculators, image processing tools, and text utilities. All tools run locally in your browser — your data never leaves your device.

Our Philosophy

🔒

Privacy First

All processing happens locally in your browser. No data is uploaded to any server.

⚡

Fast & Easy

No registration required. Apple-inspired design for a smooth user experience.

🌐

Bilingual

All tools and blog content available in both Chinese and English.

💰

Completely Free

All tools are free to use. Supported by non-intrusive advertising.

Tool Categories

  • 📐 Calculators (16) — Mortgage, Tax, BMI, Date, Housing Fund, Compound Interest & more
  • 🖼️ Image Tools (5) — Compression, Format Conversion, Resize, Base64, Color Picker
  • ✏️ Text Tools (11) — Word Counter, JSON Formatter, Base64 Encode/Decode, UUID Generator & more

Tech Stack

Built with pure static HTML/CSS/JavaScript with zero dependencies (except Chart.js for charts). Deployed on Cloudflare Pages global CDN for fast access worldwide.
