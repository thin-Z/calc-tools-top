# 文字工具站 — 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 为 calc-tools.top 增加 11 个中英双语文字处理工具，纯前端实现，隐私优先

**架构：** 独立页面模式——每个工具一个 HTML 页面（zh + en），独立 JS 文件，共享 CSS。与现有 21 个工具架构一致。中英双语通过 `zh/text/` 和 `en/text/` 路径分离，UI 模板复用现有样式（nav/header/footer）。

**技术栈：** 纯 HTML + CSS + JavaScript | 零依赖 | 所有处理在浏览器本地完成

**前置约定：**
- 中英文页面共用一套 JS 逻辑，JS 中文本地化通过 `data-i18n` 属性
- CSS 复用现有 `css/style.css` + `css/site.css`，新增 `css/text-tools.css`
- 所有文件 UTF-8 without BOM
- 每个页面包含：title、meta description、canonical、hreflang
- 文件路径遵循 `zh/text/<tool>.html` / `en/text/<tool>.html` / `js/text-tools/<tool>.js`
