### 1.2 技术栈

| 项目 | 选型 | 说明 |
|------|------|------|
| 前端 | 纯 HTML + CSS + JS | 零依赖，不打包，即改即用 |
| 图表 | Chart.js CDN | 房贷/个税等金融工具可视化 |
| 图片处理 | Canvas API | 全部本地处理，不上传服务器 |
| 文字工具 | 纯 JS 处理 | 全部浏览器本地运算 |
| 部署 | Cloudflare Pages | GitHub push 自动构建，全球 CDN |
| 域名 | calc-tools.top | Namesilo 注册，Cloudflare DNS |
| 国际化 | URL 路径 + run-time 替换 | /zh/ /en/ 分离，data-i18n 属性 |
| SEO | hreflang + JSON-LD + sitemap + robots | 搜索引擎全覆盖 |
| 字体 | Inter (Google Fonts) | Apple 设计语言风格 |

### 1.3 核心设计原则

- **隐私优先**：图片工具所有处理通过 Canvas API 在浏览器本地完成，文件永不离开用户设备
- **纯静态**：无需服务器后端，零运维成本
- **渐进增强**：搜索/分类/点赞/分页功能在不支持 JS 时优雅降级
- **暗色模式**：CSS custom properties 实现，跟随系统 / 手动切换，localStorage 持久化
- **视觉规范**：Apple 设计语言，Inter 字体，16px 圆角，毛玻璃 header，cubic-bezier 动效

## 四、核心功能

### 4.1 搜索与分类

- **搜索**：实时模糊匹配，按工具名称和描述搜索
- **分类**：8 个分类（财务/健康/生活/购物/出行/工具/图片/文字），点击过滤
- **热门搜索**：统计用户搜索词，展示"大家都在搜"热词

### 4.2 点赞系统

- 所有工具卡片、文章、详情页均支持点赞
- 共享 localStorage（key: toolbox_likes）
- 点赞常亮，刷新不丢失
- heartPop 动画（scale 1→1.3→0.9→1.1→1, 0.4s）

### 4.3 热门工具

- 综合评分算法：点赞 + 点击 + 多样性约束
- 展示前 8 个热门工具带排名 badge
- 自动降级机制：7 天无活动降级

### 4.4 暗色模式

- 跟随系统偏好（prefers-color-scheme）
- 手动切换，localStorage 持久化
- 全局 CSS transition（0.3s）

### 4.5 博客系统

- 26 篇工具指南文章（13 zh + 13 en）
- 分类标签，点击跳转首页并自动过滤
- 分页加载（默认 6 条，点击展开全部）

## 五、部署与运维

### 5.1 部署流程

1. 本地修改 → git commit
2. git push → GitHub main 分支
3. Cloudflare Pages 自动检测推送，构建部署
4. 部署完成（约 30-60 秒）

### 5.2 CI 配置

| 项目 | 值 |
|------|-----|
| 构建命令 | 无（纯静态）|
| 输出目录 | / |
| 域名 | calc-tools.top（CNAME）|
| 预览 URL | https://calc-tools-top.pages.dev |

### 5.3 网络问题处理

国内 HTTPS push 经常 `Connection was reset`，临时方案：

```bash
# 使用 GitHub REST API 推送（推荐）
# 见 docs/踩坑记录 或 update_docs.js 参考实现

# 或切换 SSH 推送
git remote set-url origin git@github.com:thin-Z/calc-tools-top.git
git push origin main
# 完成后恢复 HTTPS
git remote set-url origin https://github.com/thin-Z/calc-tools-top.git
```

### 5.4 文件编码规范

- 所有文件使用 **UTF-8 without BOM**
- 避免使用 PowerShell `Out-File` 或 `Set-Content`（产生 BOM 或 ANSI 编码损坏）
- 推荐使用 `[System.IO.File]::WriteAllBytes` 或 Node.js fs.writeFileSync

## 六、扩展指南

### 6.1 添加新计算器工具

1. **创建 HTML**：`zh/<tool-name>.html` + `en/<tool-name>.html`
2. **编写逻辑**：`js/calculators/<tool-name>.js`
3. **注册首页**：`index.html` 和 `en/index.html` 添加卡片 + JSON-LD
4. **配置分类**：`js/site.js` 的 `SITE_CONFIG.tools` 数组添加条目
5. **SEO**：`sitemap.xml` 添加 URL

### 6.2 添加新图片/文字工具

同上，路径分别为 `zh/image/` / `zh/text/`，JS 放在 `js/tools/` / `js/text-tools/`。

## 八、已知限制

| 类别 | 限制 | 原因 |
|------|------|------|
| BMI | 不适用于未成年人/孕妇/肌肉发达人群 | BMI 公式局限性 |
| 排卵期 | 仅供参考，不作为医疗建议 | 周期模型简化 |
| 图片压缩 | 不支持 SVG；GIF 动画压缩后丢失动画 | Canvas API 限制 |
| 格式转换 | BMP→JPEG/PNG 体积可能增大 | 格式特性 |
| en/text/ | keyword-density 暂缺 | 待补充英文翻译 |
| 部署 | HTTPS push 在国内网络不稳定 | GitHub 网络问题 |
