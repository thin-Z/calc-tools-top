# Calc-Tools.top — 首页直投中文版设计规格

> 创建日期: 2026-06-25
> 状态: 已批准
> 相关: [[_calculator-site_summary]] | [[建站方法论总结]]

---

## 目标

去掉根路径 `/` 的语言选择页，改为直接展示中文版首页，英文版保留在 `/en/`。

## 改动范围

### 1. 根首页 `index.html` — 替换为中文内容

| 项 | 旧值 | 新值 |
|----|------|------|
| `index.html` 内容 | 语言选择页（2个链接） | 从 `zh/index.html` 复制，调整路径和 SEO 标签 |
| CSS 路径 | — | `../css/style.css` → `css/style.css` |
| JS 路径 | — | `../js/i18n.js` → `js/i18n.js` |
| `<html lang>` | `zh-CN` | `zh-CN`（不变） |
| canonical | `https://calc-tools.top/zh/` | `https://calc-tools.top/` |
| hreflang zh-CN | `https://calc-tools.top/zh/` | `https://calc-tools.top/` |
| hreflang x-default | `https://calc-tools.top/` | `https://calc-tools.top/`（不变） |
| JSON-LD WebSite.url | `https://calc-tools.top/zh/` | `https://calc-tools.top/` |
| JSON-LD CollectionPage.url | `https://calc-tools.top/zh/` | `https://calc-tools.top/` |
| JSON-LD ItemList URLs | `https://calc-tools.top/zh/...` | `https://calc-tools.top/...` |
| 语言切换 zh 选项 | `onchange → /zh/` | `onchange → /` |

### 2. `/zh/index.html` — 改为重定向

删除原有内容，替换为：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="refresh" content="0; URL=/" />
    <title>正在跳转...</title>
    <link rel="canonical" href="https://calc-tools.top/" />
</head>
<body>
    <p><a href="/">点击这里跳转到首页</a></p>
</body>
</html>
```

### 3. 中文工具页（5个）— 更新语言切换链接

| 文件 | 改动 |
|------|------|
| `zh/mortgage.html` | 语言切换 select → zh option 的 `onchange` 值 `/zh/` → `/` |
| `zh/tax2026.html` | 同上 |
| `zh/bmi.html` | 同上 |
| `zh/date-calc.html` | 同上 |
| `zh/housing-fund.html` | 同上 |

### 4. 英文工具页（6个）— 更新 hreflang 中文 alternate

| 文件 | 改动 |
|------|------|
| `en/index.html` | hreflang zh-CN alternate: `/zh/` → `/` |
| `en/mortgage.html` | 同上（工具页的 alternate URL 保持 `/zh/mortgage` 等不变，仅首页相关改） |

检查：英文工具页的 hreflang zh-CN 指向的是对应工具的 `/zh/xxx` 路径，**非首页 URL**，所以实际上英文工具页不需要改动。仅 `en/index.html` 需要改。

### 5. `js/i18n.js` — 更新语言切换

```
switchLang('zh') 跳转目标: /zh/ → /
```

### 6. `sitemap.xml` — 去除 `/zh/` 条目，更新根页 alternate

- 移除 `<url><loc>https://calc-tools.top/zh/</loc>...</url>` 条目
- 根页的 alternate zh-CN 从 `/zh/` 改为 `/`
- 其余工具条目不受影响（仍保持 `/zh/mortgage` 等路径）

---

## 不受影响的范围

- `/en/` 目录下所有页面 — 不修改
- 各工具 JS 计算逻辑 — 不修改
- `css/style.css` — 不修改
- `vercel.json` — 不修改
- `robots.txt` — 不修改
- 各工具页的 canonical — 保持 `/zh/xxx` 不变

## 影响检查

| 检查项 | 结论 |
|--------|------|
| 旧书签 `/zh/` | 可访问（自动跳转到 `/`）|
| 旧外链指向 `/zh/xxx` 工具页 | 不受影响，路径不变 |
| SEO hreflang 一致性 | 根页作为 zh-CN canonical，/en/ 作为 en canonical |
| 语言切换功能 | `/` 和 `/en/` 之间切换正常 |
| JSON-LD 结构化数据 | URL 已更新为根路径 |
| sitemap | 去除重复条目，提高 SEO 质量 |
