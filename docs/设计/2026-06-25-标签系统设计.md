# 工具标签 + 文章标签系统 — 设计规格

> 创建日期: 2026-06-25
> 状态: 已批准
> 相关: [[_calculator-site_summary]]

---

## 目标

1. 工具卡片显示所属分类标签
2. 文章按标签分类，详情页显示标签，支持按标签过滤
3. 创建文章归档页

## 改动范围

### 1. 工具卡片标签

每个工具卡片底部添加分类标签，样式为彩色圆角药丸。

**标签样式映射：**

| 分类 | CSS 类 | 背景色 | 文字色 |
|------|--------|--------|--------|
| 💰 财务 | `.tag-finance` | `#eef2ff` | `#4f46e5` |
| 🏥 健康 | `.tag-health` | `#f0fdf4` | `#16a34a` |
| 🏠 生活 | `.tag-life` | `#fefce8` | `#ca8a04` |
| 🛒 购物 | `.tag-shopping` | `#fdf2f8` | `#db2777` |
| 🚗 出行 | `.tag-travel` | `#f0f9ff` | `#0284c7` |
| 🔧 工具 | `.tag-utility` | `#f5f3ff` | `#7c3aed` |

**HTML 结构（以房贷计算器为例）：**

```html
<div class="tool-card-wrap">
  <a href="..." class="tool-card" data-category="finance">
    <div class="icon">🏠</div>
    <h3>房贷计算器</h3>
    <p>...</p>
  </a>
  <button class="like-btn">...</button>
  <div class="tool-tags">
    <a href="/" class="tag tag-finance" data-tag="finance">💰 财务</a>
  </div>
</div>
```

**多标签示例（折扣计算器）：**

```html
<div class="tool-tags">
  <a href="/" class="tag tag-shopping" data-tag="shopping">🛒 购物</a>
  <a href="/" class="tag tag-finance" data-tag="finance">💰 财务</a>
</div>
```

**标签点击行为：** 点击标签回到首页，自动选中对应分类，过滤工具列表。

### 3. 文章归档页

新建 `/blog/index.html`（中英文共用模板，通过语言切换路由）。

**归档页结构：**

```
<head>
  - title / meta description
  - canonical / hreflang
  - CSS 引用
</head>
<body>
  <header> (统一导航) </header>
  <main>
    <h1>📖 全部文章</h1>
    <!-- 标签筛选器 -->
    <div class="category-filters">
      <button class="category-chip active" data-category="all">全部</button>
      <button class="category-chip" data-category="finance">💰 财务</button>
      ...
    </div>
    <!-- 文章列表 -->
    <div class="article-list">
      <article class="article-item" data-category="finance">
        <h2><a href="/blog/zh/equal-installment-vs-equal-principal.html">等额本息 vs 等额本金怎么选</a></h2>
        <p class="article-meta">2026-06-25</p>
        <p class="article-summary">...</p>
        <div class="article-tags">
          <a href="/blog/" class="tag tag-finance" data-tag="finance">💰 财务</a>
        </div>
      </article>
      ...
    </div>
  </main>
  <footer>...</footer>
</body>
```

**文章分类映射：**

| 文章 | 分类 |
|------|------|
| 等额本息 vs 等额本金怎么选 | finance |
| 2026 房贷利率趋势解读 | finance |
| 2026 个税抵扣全攻略 | finance |
| BMI 多少算正常 | health |
| 公积金贷款额度怎么算 | finance |
| 日期计算实用技巧 | life |

**归档页标签过滤：** 复用 `site.js` 中的分类过滤逻辑。

### 5. site.js 更新

在 `js/site.js` 中：

- 标签点击处理：拦截标签链接的默认跳转，触发对应分类筛选
- 文章归档页的分类过滤（复用已有逻辑，扩展支持 `.article-item`）

### 6. sitemap 更新

在 `sitemap.xml` 新增 `/blog/` 首页条目。

## 不受影响的范围

- 各工具页面的内容 — 不修改
- 点赞系统 — 不修改
- 搜索功能 — 不修改
- P1-P4 的其余计划 — 不修改

## 影响检查

| 检查项 | 结论 |
|--------|------|
| 首页工具卡片布局 | 标签在卡片底部，不影响 grid 布局 |
| 点赞按钮位置 | 标签在底部右下，与点赞（右上）不冲突 |
| 文章页 SEO | 新增标签不影响 Article Schema |
| 多语言 | 中英文标签颜色一致，文字用对应语言 |
| 移动端适配 | 标签自动换行，不影响布局 |
