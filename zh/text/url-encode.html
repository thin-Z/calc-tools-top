什么是 URL 编码？

                
URL 编码（又称 Percent-encoding）是一种将 URL 中的特殊字符和非 ASCII 字符转换为安全格式的编码方式。它使用百分号（%）后跟两位十六进制数来表示字符的字节值。URL 编码确保数据在 HTTP 协议中安全传输，避免特殊字符破坏 URL 结构。

                
本工具使用浏览器内置的 encodeURIComponent 和 decodeURIComponent API 实现 UTF-8 安全编解码，确保中文、日文、韩文等非 ASCII 字符在编解码过程中不会出现乱码问题。所有处理均在浏览器本地完成，数据不会上传至任何服务器。


                
URL 编解码的常见场景

                
    
                    
  • GET 请求参数 — 对查询参数进行 URL 编码，确保特殊字符和中文安全传输
    • 
                    
  • POST 表单数据 — application/x-www-form-urlencoded 格式需要 URL 编码
    • 
                    
  • AJAX / Fetch 请求 — 构建安全的 API 请求 URL 参数
    • 
                    
  • URL 重定向 — 在重定向 URL 中编码目标地址
    • 
                    
  • Cookie 值编码 — 对特殊字符进行 URL 编码以便安全存储
    • 
                


                
注意事项

                
encodeURIComponent 与 encodeURI 不同：encodeURI 不编码 URL 中合法的特殊字符（如 : // ? #），而 encodeURIComponent 会编码所有非字母数字字符，包括这些特殊字符。本工具使用 encodeURIComponent / decodeURIComponent，适用于编码 URL 查询参数值以确保安全。
