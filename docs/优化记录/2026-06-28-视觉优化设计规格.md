## 二、字体系统

### 2.1 字体栈

主要字体：**Inter** (Google Fonts)
回退方案：`-apple-system, BlinkMacSystemFont, sans-serif`

加载方式：
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

## 三、组件设计

### 3.1 工具卡片

- 背景：白色（浅色）/ #2C2C2E（深色）
- 圆角：**16px**（从 12px 升级）
- 阴影：`0 1px 3px rgba(0,0,0,0.06)` 
- 内边距：20px
- 图标容器：44×44px，圆角 12px，分类色渐变背景
- hover: translateY(-4px) + 强化阴影 0.3s cubic-bezier

### 3.2 分类 Chip (category-chip)

- 圆角：999px
- 默认状态：背景 #F0F0F0，文字 #555
- 活跃状态：品牌蓝渐变背景 `linear-gradient(135deg, #007AFF, #0056CC)` + 白色文字 + `box-shadow: 0 2px 8px rgba(0,122,255,0.3)`
- hover: scale(1.05) 放大动效

### 3.3 搜索框

- 圆角：12px
- 边框：1.5px solid #E5E5EA
- 背景：`rgba(255,255,255,0.8)` + `backdrop-filter: blur(10px)`
- focus: 边框变品牌蓝 + `box-shadow: 0 0 0 3px rgba(0,122,255,0.15)`
- 左侧放大镜图标

### 3.4 点赞按钮

- 样式：pill 形状 `border-radius: 999px`
- 边框：1.5px solid #E5E5EA
- 默认：灰色文字和边框
- hover：边框变红 `#FF6B6B` + 文字变红
- liked 状态：红色填充背景 `#FFF5F5` + 红色边框
- 点击动画：爱心弹出 (scale 1→1.3→0.9→1.1→1, 0.4s)

### 3.5 热门工具卡片

- 继承普通卡片样式
- 左上角排名 badge：`linear-gradient(135deg, #F59E0B, #EF4444)`
- 右上角点赞数
- 图标容器背景：橙色系渐变 `linear-gradient(135deg, #FFFBEB, #FEF3C7)`
- 边框：1.5px solid #FEF3C7

## 四、Header 毛玻璃效果

- 固定顶部
- 滚动时背景：`rgba(255,255,255,0.72)` + `backdrop-filter: blur(20px)`
- 深色模式：`rgba(0,0,0,0.72)` + `backdrop-filter: blur(20px)`
- 底部边框：1px solid `rgba(0,0,0,0.06)`（浅色）/ `rgba(255,255,255,0.06)`（深色）

## 五、交互动效

### 5.1 P0 核心动效

| 元素 | 动效 | 实现 | 参数 |
|------|------|------|------|
| 工具卡片 hover | 上浮 + 阴影 | transition + transform | 0.3s cubic-bezier(0.25,0.46,0.45,0.94) |
| 分类 Chip hover | 放大 | transition + transform: scale | 0.2s ease |
| 点赞点击 | 爱心弹出 | @keyframes heartPop | 0.4s ease |
| 搜索框 focus | 蓝色发光 | box-shadow transition | 0.2s ease |
| 按钮点击 | 缩小反馈 | transform: scale(0.95) | 0.15s ease |
| 分类过滤 | 卡片淡入淡出 | opacity transition | 0.2s ease |

### 5.2 P1 增强动效

| 元素 | 动效 | 实现 |
|------|------|------|
| 暗色切换 | 图标旋转淡入 | keyframes spinIn, 0.3s |
| 计算结果 | 数字渐入 | fadeIn + translateY |

## 七、设计原则

1. **性能优先**：全部使用 CSS，零 JS 动画库，零额外网络请求（Inter 字体 20KB 可接受）
2. **渐进增强**：不支持 backdrop-filter 的浏览器回退到纯色背景
3. **一致性**：所有组件共用同一套圆角/阴影/色彩变量体系
4. **暗色优先**：设计时同时考虑浅色和深色模式，不事后适配
