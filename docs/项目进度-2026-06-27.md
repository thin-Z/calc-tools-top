## 踩坑记录

### 编码
- PowerShell Set-Content / Out-File 默认 ANSI 编码，静默破坏 UTF-8 字符
- 正确做法：[System.IO.File]::WriteAllBytes() + [Text.Encoding]::UTF8.GetBytes()
- atob() 解码 GitHub API base64 时不处理 UTF-8 多字节字符
- 正确做法：TextDecoder("utf-8").decode(new Uint8Array(...))

### GitHub API
- raw.githubusercontent.com 国内访问频繁超时
- 改用 api.github.com/git/blobs + base64 解码
- "Update is not a fast forward" — 必须先从 refs/heads/main 获取最新 SHA 作为 parent
