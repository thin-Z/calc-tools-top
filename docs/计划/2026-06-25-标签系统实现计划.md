### 任务 2：JS — 标签点击处理 + 归档页分类过滤

**文件：** `calculator-site/js/site.js`

- [ ] **步骤 1：添加标签点击处理**

在 `initCategoryFilters` 函数后添加标签点击初始化：

```javascript
function initTagClicks() {
    document.querySelectorAll('.tag[data-tag]').forEach(tag => {
        tag.addEventListener('click', (e) => {
            e.preventDefault();
            const cat = tag.dataset.tag;
            // Navigate to homepage and trigger category filter
            const homeUrl = tag.closest('[lang="zh-CN"]') ? '/' : '/en/';
            sessionStorage.setItem('preselectCategory', cat);
            window.location.href = homeUrl;
        });
    });
}

// In DOMContentLoaded:
initTagClicks();
```

- [ ] **步骤 2：首页加载时检查预选分类**

```javascript
// At the top of initCategoryFilters:
const preselect = sessionStorage.getItem('preselectCategory');
if (preselect) {
    sessionStorage.removeItem('preselectCategory');
    setTimeout(() => {
        const chip = document.querySelector(`.category-chip[data-category="${preselect}"]`);
        if (chip) chip.click();
    }, 100);
}
```
